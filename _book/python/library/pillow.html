Pillow

Pillow

一个功能强大却又简单易用的图像处理模块。

与 PIL 的关系： Pillow is the friendly PIL fork by Alex Clark and Contributors. PIL is the Python Imaging Library by Fredrik Lundh and Contributors.

PIL 仅支持到 Python2.7，且年久失修，于是一群志愿者在 PIL 的基础上创建了兼容的版本，就是 Pillow。

Pillow 和 PIL 不能共存于一个环境中

  • 官方文档
  • 项目地址

Pillow 各版本均支持 Python2.7，不同 Pillow 版本支持不同的 3.x 版本，详细情况要查看官方文档。

安装 (官方文档安装部分写的非常详细非常全面)

pip install Pillow
